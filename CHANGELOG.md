# RegexBench Changelog

## [0.1.0] - 2026-07-05

### Added
- Regex parser with full syntax support
- NFA construction (Thompson's algorithm)
- DFA construction (subset construction)
- DFA minimization (Hopcroft's algorithm)
- Backtracking matching engine
- Test runner with pass/fail reporting
- Benchmark runner with timing statistics
- Pattern analyzer for backtracking risk detection
- Code exporter for Python, JavaScript, Go, Rust, Java, C#
- NFA/DFA visualization
- Match timeline visualization
- Interactive REPL mode
- JSON output format support
